AVANCE 02 – PROYECTO FINAL
Interfaces y funcionalidad a implementar para la presentación del segundo avance del proyecto.
1.

GESTIÓN DE VEHÍCULOS
a.

Vehículos
1)

Lista de Vehículos
Muestra el listado de vehículos registrados. Desde esta interface se puede crear nuevos
vehículos, editarlos o eliminarlos.

2)

Nuevo Vehículo
Permite agregar nuevos vehículos. Se debe ingresar: código, tipo de vehículo, nombre
del vehículo, placa, año, color, marca, modelo, capacidad de carga (Tn), capacidad de
combustible (L), capacidad de compactación (Tn), capacidad de personas (Esto
determinará cuantas personas se podrá asignar a los vehículos), descripción y estado.

3)

Carga de Imágenes
Permite agregar varias imágenes a cada vehículo. Se puede hacer click en esta sección
y buscar las imágenes desde la computadora o también se pueden arrastrar las
imágenes hasta esta área.

Se debe guardar para que las imágenes se agreguen al vehículo.

2.

GESTIÓN DE PERSONAL
a.

Tipo de Personal
1)

Lista de Tipos de Personal
Muestra el listado de tipos de personal registrado. Desde esta interface se puede crear
nuevos tipos de personal, editarlos o eliminarlos.

2)

Nuevo Tipo de Personal
Permite agregar nuevos tipos de personal. Se debe ingresar nombres y descripción.

b.

Personal
1)

Lista de Personal
Muestra el listado de personal registrado. Desde esta interface se puede crear nuevo
personal, editarlo o eliminarlo.

2)

Nuevo Personal
Permite agregar nuevo personal. Se debe ingresar: DNI, tipo de personal, nombres,
apellidos, fecha de nacimiento, teléfono, email, estado, contraseña, dirección y una foto.

c.

Contratos
1)

Lista de Contratos
Muestra el listado de contratos registrados. Desde esta interface se puede crear nuevos
contratos, editarlos o desactivarlos.

2)

Nuevo Contrato:
Permite agregar nuevo contrato al personal. Se debe ingresar: el personal, el tipo de
contrato (Permanente, nombrado, temporal). Si el contrato es permanente o nombrado,
solo se coloca la fecha de inicio del contrato, si es temporal se coloca la fecha de inicio
y finalización del contrato. También se deberá ingresar el salario y de forma opcional el
período de prueba.
Los contratos determinarán qué personal estará disponible para ser asignado en las
programaciones de rutas, ya sea como conductor o ayudante.
